# Maker Time Protector

基于 **Cal Newport《Deep Work》** 和 **Time Blocking** 方法的智能时间管理技能。

## 核心理念

- **区分 Maker Time 和 Manager Time**：保护深度工作的大块时间
- **主动设计理想周**：而不是被动响应日程
- **保护专注时间**：对干扰说"不"

## 安装

本技能位于 `src/agent/prompts/skills/maker-time-protector/`

如需安装到全局技能目录：

```bash
cp -r src/agent/prompts/skills/maker-time-protector ~/.agents/skills/
```

## 触发短语

- "设计我的一周"
- "理想周模板"
- "保护专注时间"
- "创建深度工作块"
- "分析我的时间块"
- "这周 Maker Time 够吗"
- "时间块复盘"

## 主要功能

1. **设计理想周**：根据你的习惯和固定承诺，生成每周 Maker Time 模板
2. **每日时间块规划**：在空闲时间中识别并创建深度工作块
3. **Maker Time 分析**：统计本周专注时间，对比推荐值
4. **时间块保护**：提供拒绝干扰的话术模板
5. **周度复盘**：分析执行情况，给出改进建议

## 默认设置

针对用户习惯（晚睡晚起、计算机系学生）优化：

- **下午块**：14:00 - 18:00（4 小时）
- **晚上块**：20:00 - 24:00（4 小时）
- **深夜块**（可选）：00:00 - 02:00（2 小时）

## 使用示例

```
用户：帮我设计这周的理想周模板

助手：好的！先确认一下你这周的固定承诺...
```
