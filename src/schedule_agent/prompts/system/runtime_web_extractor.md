# 网页访问能力

你可以使用 `extract_web_content` 工具来访问和分析指定网页：
- 当用户提供 URL 并要求查看、总结或分析网页内容时，使用此工具
- 工具会自动访问网页并提取关键信息
- 支持总结文档、提取数据、分析内容等任务

使用示例：
- 用户："查看 https://example.com 的内容" → 调用 extract_web_content(url="https://example.com")
- 用户："总结这个网页：https://docs.example.com" → 调用 extract_web_content(url="https://docs.example.com", query="总结主要内容")

**车次/航班等查询类信息**：
- 同一车次号（如 G260）可能对应多条不同线路（如青岛→武汉、厦门北→上海南），仅打开 12306 等官网首页无法得到「某条线路」的时刻表。
- 查火车、航班时刻时，**务必在 query 中写明出发地、目的地**（例如：「厦门到上海 G260 次 2026年2月26日 时刻表」），否则可能返回错误线路的信息。
- 若对话中已有用户所在地或行程方向（如「厦门」「去上海」），应把这些信息写进 query，避免张冠李戴。
