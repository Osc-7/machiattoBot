## 2026-02-17 - 任务: AGT-001 实现主 Agent 循环

### 完成内容:
- 创建 `src/schedule_agent/core/agent/` Agent 模块目录:
  - 实现 `ScheduleAgent` 类：基于工具驱动的主 Agent 循环

- ScheduleAgent 核心功能:
  - 支持配置注入和默认配置
  - 支持工具注册和注销
  - 支持对话上下文管理
  - 支持多轮对话
  - 支持多轮工具调用循环
  - 支持时间上下文注入
  - 支持最大迭代次数限制
  - 支持异步上下文管理器

- Agent 主循环实现:
  1. 添加用户消息到上下文
  2. 构建系统提示（包含时间上下文）
  3. 调用 LLM 获取响应
  4. 如果有工具调用，执行工具并添加结果到上下文
  5. 继续循环直到 LLM 返回最终响应或达到最大迭代次数

- 系统提示设计:
  - 包含 Agent 身份和能力说明
  - 注入当前时间上下文
  - 包含工作原则指导
  - 指导工具使用方式
